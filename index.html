Projects

Telco Customer Churn Analysis

A machine learning project combining logistic regression and K-means clustering to identify high-risk churn customers and uncover key drivers such as contract type, tenure, and payment methods. Used a dataset of 7,043 entries from Kaggle.

  • 📊 Logistic Regression Model (AUC: 0.84) for churn prediction
  • 🔍 Identified top risk factors: Month-to-month plans, fiber-optic service, e-check payment
  • 📈 K-Means Clustering segmented customers into high/low-risk groups (Silhouette ≈ 0.25)
  • 💡 Actionable retention strategies: contract upgrades, bundling services, autopay incentives

U.S. Homicide Data Dashboard (Power BI)

An interactive Power BI dashboard analyzing U.S. homicide trends from 1976 to 2023. This project explores patterns in weapon usage, victim/offender demographics, and regional crime trends using visual analytics.

  • 📌 Covers nearly 60 years of FBI-reported homicide cases
  • 📍 Includes filters by year, state, victim/offender profile, and weapon type
  • 📊 Designed with dynamic visuals, slicers, and drill-through navigation
  • 🔗 Deployed publicly via Power BI for online interactivity
